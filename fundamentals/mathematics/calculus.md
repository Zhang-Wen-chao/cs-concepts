# 微积分 - Calculus

> 研究变化率和累积量，是优化和连续系统建模的基础

## 📚 核心概念体系

### 1. 极限 (Limit)
函数在某点附近的行为趋势：$\lim_{x \to a} f(x) = L$

| 重要工具 | 说明 |
|---------|------|
| ε-δ 定义 | $0 < \|x-a\| < \delta \implies \|f(x)-L\| < \epsilon$ |
| 洛必达法则 | $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 时，$\lim \frac{f}{g} = \lim \frac{f'}{g'}$ |
| 重要极限 | $\lim_{x\to0} \frac{\sin x}{x} = 1$, $\lim_{n\to\infty} (1+\frac{1}{n})^n = e$ |

### 2. 导数 (Derivative)
瞬时变化率：$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

| 规则 | 公式 |
|------|------|
| 幂函数 | $(x^n)' = nx^{n-1}$ |
| 链式法则 | $(f(g(x)))' = f'(g(x)) \cdot g'(x)$ |
| 乘法法则 | $(fg)' = f'g + fg'$ |

**应用**：梯度下降、反向传播、物理仿真

### 3. 积分 (Integral)
累积量（面积）：$\int_a^b f(x) dx$

**微积分基本定理**：$\int_a^b f(x)dx = F(b) - F(a)$，其中 $F'(x) = f(x)$

**应用**：概率密度函数的归一化、曲线长度、物理量积累

### 4. 优化 (Optimization)
- **极值**：$f'(x) = 0$（临界点），通过 $f''(x)$ 判断极大/极小
- **拉格朗日乘数法**：带约束优化 $\nabla f = \lambda \nabla g$
- **数值优化**：梯度下降、牛顿法

### 5. 微分方程 (DE)
描述变化率与状态关系的方程：$\frac{dy}{dx} = f(x, y)$
