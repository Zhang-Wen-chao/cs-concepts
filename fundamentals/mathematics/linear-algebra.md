# 线性代数 - Linear Algebra

> 向量空间和线性变换，现代计算机科学的核心数学工具

## 📚 核心概念体系

### 1. 向量 (Vectors)
- **定义**：有序数组 $\vec{v} = (v_1, v_2, ..., v_n)$
- **运算**：加法、数乘、点积 $\vec{a} \cdot \vec{b} = \|a\|\|b\|\cos\theta$
- **几何**：方向 + 大小，点积衡量投影关系
- **应用**：特征表示、语义嵌入、权重向量

### 2. 矩阵 (Matrices)
- **定义**：二维数组 $A_{m \times n}$
- **运算**：乘法、转置、逆矩阵
- **几何意义**：线性变换（旋转、缩放、投影）
- **特殊矩阵**：单位矩阵、对角矩阵、对称矩阵、正交矩阵

### 3. 线性方程组
- **高斯消元法**：行变换化简为行阶梯形
- **矩阵形式**：$Ax = b$
- **解的存在性**：$rank(A) = rank([A|b])$
- **应用**：最小二乘法、系统建模

### 4. 向量空间 (Vector Spaces)
- **基与维数**：基 = 一组线性无关的"坐标轴"，维数 = 基的元素个数
- **线性无关 vs 线性相关**：能否互相表示
- **子空间**：大的空间里的"小空间"
- **秩 (Rank)**：矩阵列空间/行空间的维数

### 5. 特征值与特征向量
- **定义**：$Av = \lambda v$ — 变换后方向不变，只缩放
- **几何**：特征向量 = 变换的"主方向"，特征值 = 拉伸倍数
- **应用**：PCA 降维、PageRank、图像压缩、谱聚类

### 6. 矩阵分解

| 分解 | 形式 | 应用 |
|------|------|------|
| **SVD** | $A = U\Sigma V^T$ | 降维、推荐系统、压缩 |
| **QR** | $A = QR$ | 数值计算、最小二乘 |
| **特征分解** | $A = Q\Lambda Q^{-1}$ | 对称矩阵对角化 |

## 🔗 与 CS 的交叉

| CS 领域 | 线性代数概念 |
|---------|------------|
| 神经网络 | 矩阵乘法 = 线性层，权重矩阵 |
| 图形学 | 旋转/缩放/投影矩阵 |
| 推荐系统 | 矩阵分解 = Embedding |
| 搜索引擎 | 特征向量 = PageRank |
| 计算机视觉 | SVD 降噪、PCA |
| 量子计算 | 向量 = 量子态，矩阵 = 门操作 |
