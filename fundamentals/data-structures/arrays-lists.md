# Arrays and Lists - 数组与列表

> 最基础的数据结构：如何存储有序的数据集合？

## 数组 (Array)
**定义**：在连续内存中存储相同类型元素的集合。

*   **核心特性**：
    *   **随机访问 O(1)**：通过 `起始地址 + 索引 × 元素大小` 计算得出。
    *   **缓存友好**：内存连续，对 CPU 缓存极为友好。
    *   **限制**：大小通常固定，插入/删除需移动元素 (O(n))。

## 动态数组 (Dynamic Array)
**定义**：能够自动扩容的数组（如 Python 的 `list` 或 Java 的 `ArrayList`）。

*   **原理**：当容量满时，分配一个更大的新空间（通常翻倍），将旧元素复制过去。
*   **摊销分析**：虽然扩容是 O(n)，但由于扩容次数极少，平均每次添加操作仍为 O(1)。

## 链表 (Linked List)
**定义**：由节点组成的序列，每个节点包含数据和指向下一个节点的指针。

*   **核心特性**：
    *   **动态扩展**：无需预分配内存，指针连接即可。
    *   **插入/删除 O(1)**：只需修改相邻节点的指针指向。
    *   **访问慢 O(n)**：必须从头遍历查找，缓存不友好。

## 数组 vs 链表 (选择速查)

| 特性 | 数组 | 链表 |
| :--- | :--- | :--- |
| **内存布局** | 连续 | 分散 |
| **访问** | **O(1)** | O(n) |
| **插入/删除** | O(n) | **O(1)** |
| **缓存性能** | **优** | 差 |
