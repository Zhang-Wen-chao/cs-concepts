# Physical and Data Link Layers - 物理层与数据链路层

> 物理介质上如何传比特？帧如何确保可靠？

## 🎯 物理层 (Physical Layer)

物理层负责在物理介质上传输原始的比特流（0 和 1），定义电压、频率、接口等电气特性。

### 传输介质对比

| 介质 | 原理 | 速度 | 距离 | 优/缺点 |
|---|---|---|---|---|
| **双绞线** | 两根铜线绞合抵消干扰 | Cat 5e: 1Gbps, Cat 6: 10Gbps | ~100m | ✅ 便宜易装 ❌ 距离短 |
| **同轴电缆** | 中心导线+屏蔽层 | 可达 Gbps | 较远 | ✅ 抗干扰 ❌ 硬且贵 |
| **光纤** | 光的全反射 | Tbps 级 | 数十公里 | ✅ 带宽高/抗干扰 ❌ 贵/安装复杂 |
| **无线** | 电磁波 (Wi-Fi/蓝牙) | 802.11ax: 9.6Gbps | 几十米 | ✅ 灵活 ❌ 干扰大/安全性差 |

### 数字信号编码

- **NRZ (非归零编码)**：高电平 = 1，低电平 = 0
- **曼彻斯特编码**：上跳 = 0，下跳 = 1（自带时钟信号）

## 🎯 数据链路层 (Data Link Layer)

```
功能：
1. 成帧 (Framing)：比特流 → 帧
2. 物理寻址：MAC 地址 (48位)
3. 错误检测：CRC、奇偶校验
4. 访问控制：CSMA/CD、CSMA/CA
5. 流量控制：防止接收方过载
```

### 以太网帧结构

```
┌──────┬─────────┬─────────┬──────┬────────┬─────┐
│前导码 │目标MAC  │ 源MAC   │类型  │ 数据   │ FCS │
│ 8B   │  6B     │  6B     │ 2B   │46-1500B│ 4B  │
└──────┴─────────┴─────────┴──────┴────────┴─────┘
- 前导码：同步
- 类型：0x0800 = IP
- FCS：CRC 校验
```

### MAC 地址

| 字段 | 长度 | 说明 |
|---|---|---|
| OUI (Organizationally Unique Identifier) | 24 位 (3B) | 厂商标识, e.g. 00:1A:2B = Intel |
| NIC (Network Interface Controller) | 24 位 (3B) | 网卡编号 |
| 广播地址 | FF:FF:FF:FF:FF:FF |

## 🔄 介质访问控制 (MAC)

### 信道分配方式

| 方式 | 子类 | 描述 |
|---|---|---|
| **静态分配** | FDMA/TDMA/CDMA | 频分/时分/码分，简单但效率低 |
| **动态分配** | CSMA/CD (以太网) | 先听后发，边发边听，冲突退避 |
| 动态分配 | CSMA/CA (Wi-Fi) | RTS/CTS 握手避免无线冲突 |

### CSMA/CD 工作流程

```
载波侦听 → 信道空闲 → 发送数据 → 冲突检测
                                  ↓
                              检测到冲突 → 停止发送 → 随机退避 → 重试
```

---

## ❌ 错误检测方法对比

| 方法 | 原理 | 检测能力 | 开销 |
|---|---|---|---|
| **奇偶校验** | 加 1 位使 1 的个数为奇/偶 | 只能检测奇数位错误 | 1 bit |
| **CRC** (循环冗余校验) | 多项式除法求余数 | 强（检测突发错误） | 2-4 字节 |
| 校验和 (Checksum) | 数据求和取反 | 较弱 | 2 字节 |

---

## 🔀 交换机 (数据链路层)

| 设备 | 层次 | 转发依据 |
|---|---|---|
| **集线器 (Hub)** | 物理层 | 广播到所有端口 |
| **交换机 (Switch)** | 数据链路层 | MAC 地址表 → 精确转发 |
| 路由器 | 网络层 | IP 路由表 |

**交换机工作流程**：学习源 MAC 到端口 → 查目标 MAC → 已知则单播转发 / 未知则泛洪

## 🔗 概念关系图

```
        比特流 (Bits)
            ↓
 ┌─────────────────────┐
 │  物理层 (Physical)  │  ← 介质：双绞线 / 光纤 / 无线
 │  传输介质、编码调制  │  ← 编码：NRZ / 曼彻斯特
 └─────────┬───────────┘
           ↓ 组装成帧
 ┌─────────────────────┐
 │ 数据链路层 (Datalink)│  ← MAC 地址：48位 OUI+NIC
 │ 成帧·寻址·错误检测   │  ← CSMA/CD / CSMA/CA
 │ 介质访问控制         │  ← 交换机：MAC 表学习转发
 └─────────────────────┘
           ↓ 帧 (Frame) → 网络层
```
