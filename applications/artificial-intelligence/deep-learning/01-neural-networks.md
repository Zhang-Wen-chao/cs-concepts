# 神经网络

## 一句话

把多个线性模型叠起来，中间加非线性激活函数，就能拟合任意复杂函数。

---

## 从线性到非线性

### 线性模型的局限

逻辑回归只能画直线决策边界。但真实世界的数据往往是这样的：

```
x₂ ↑                x₂ ↑
   |  ○  ○  ●         |  ○  ○  ●
   |  ○  ●  ●         |  ○  ●  ●
   |  ●  ●  ○         |  ●  ●  ○
   └────────→ x₁     └────────→ x₁
   直线可分             不可分（圆圈形）
```

### 神经网络怎么做

加一层**隐藏层** + **非线性激活函数**：

```
输入 x (2维) → 隐藏层(16个神经元) → ReLU → 输出层(2分类)
```

隐藏层的每个神经元 = 一个线性模型 + 激活函数。多个神经元组合起来，就能画任意形状的决策边界。

---

## 网络结构

```
输入层        隐藏层          输出层
  x₁ ────→ n₁ ──→ n₁ ──→ n₁
  x₂ ────→ n₂ ──→ n₂ ──→ n₂       → ŷ
  ...      ...     ...     ...
  x_d ───→ n_h ──→ n_h ──→ n_h
```

### 数学表达

```
z₁ = W₁·x + b₁        # 线性变换
a₁ = ReLU(z₁)          # 非线性激活
z₂ = W₂·a₁ + b₂        # 第二层线性变换
a₂ = σ(z₂)             # 输出层激活（sigmoid/softmax）
ŷ = a₂                 # 预测
```

- W₁ 形状 `[hidden, input_dim]`
- W₂ 形状 `[output_dim, hidden]`

**每多一层，模型能表达的函数就越复杂。**

---

## 为什么需要激活函数？

如果没有激活函数（或者用线性激活）：

```
z₂ = W₂·(W₁·x + b₁) + b₂ = (W₂·W₁)·x + (W₂·b₁ + b₂)
```

两层线性 = 一层线性。叠再多层还是直线。

激活函数（ReLU、Sigmoid、Tanh）引入**非线性**，让深层网络真正"深"起来。

| 激活函数 | 公式 | 特点 |
|---|---|---|
| ReLU | max(0, x) | 简单、缓解梯度消失、LLM 标配 |
| Sigmoid | 1/(1+e⁻ˣ) | 输出 0~1，用于二分类输出层 |
| Tanh | (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | 输出 -1~1，常用于 RNN |
| GELU | x·Φ(x) | 比 ReLU 平滑，GPT/BERT 用 |

---

## 万能近似定理

> 一个足够宽的隐藏层 + 非线性激活，可以逼近任意连续函数。

这解释了为什么神经网络这么强——理论上它可以学任何映射关系。
