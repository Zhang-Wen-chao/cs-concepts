# 反向传播

## 一句话

从输出层的损失出发，**沿计算图倒着走**，用链式法则算出每个参数对损失的贡献（梯度），然后更新参数。

---

## 链式法则

如果 `L = f(g(h(x)))`，那么：

```
∂L/∂x = ∂f/∂g · ∂g/∂h · ∂h/∂x
```

从外层一层层往里乘。

---

## 一个具体例子

最简单的网络：`ŷ = w·x + b`，损失 `L = (ŷ - y)²`

### 前向传播

```
z = w·x + b      # 线性层
ŷ = z            # 无激活（线性回归）
L = (ŷ - y)²     # MSE 损失
```

### 反向传播（求梯度）

```
∂L/∂ŷ = 2(ŷ - y)
∂ŷ/∂z = 1
∂z/∂w = x
∂z/∂b = 1

∂L/∂w = ∂L/∂ŷ · ∂ŷ/∂z · ∂z/∂w = 2(ŷ - y) · 1 · x
∂L/∂b = ∂L/∂ŷ · ∂ŷ/∂z · ∂z/∂b = 2(ŷ - y) · 1 · 1
```

### 更新

```
w = w - η · ∂L/∂w
b = b - η · ∂L/∂b
```

## 多层网络的反向传播

```
输入 x → 隐藏层(ReLU) → 输出层(sigmoid) → 损失 L
```

```
前向：
z₁ = W₁·x + b₁
a₁ = ReLU(z₁)
z₂ = W₂·a₁ + b₂
ŷ = σ(z₂)
L = -(y·log(ŷ) + (1-y)·log(1-ŷ))

反向（从后往前）：
∂L/∂z₂ = ŷ - y                              # sigmoid + 交叉熵的简洁形式
∂L/∂W₂ = ∂L/∂z₂ · a₁ᵀ                       # 输出层权重梯度
∂L/∂b₂ = ∂L/∂z₂                              # 输出层偏置梯度
∂L/∂a₁ = W₂ᵀ · ∂L/∂z₂                       # 误差传到隐藏层输出
∂L/∂z₁ = ∂L/∂a₁ · ReLU'(z₁)                 # ReLU 求导（z₁>0 时为 1，否则 0）
∂L/∂W₁ = ∂L/∂z₁ · xᵀ                        # 隐藏层权重梯度
∂L/∂b₁ = ∂L/∂z₁                              # 隐藏层偏置梯度
```

关键：**W 的梯度 = 上层传来的误差 × 本层输入**。

---

## 计算图可视化

```
x ──→ (W₁·x+b₁) ──→ ReLU ──→ (W₂·a₁+b₂) ──→ σ ──→ L
                         ↑                    ↑
                      误差从这              误差从这
                      往左传                往左传
```

框架（PyTorch/TensorFlow）自动维护计算图，自动求导。你只要写前向，反向自动算。

---

## 梯度消失 vs 梯度爆炸

| 问题 | 现象 | 原因 | 解决 |
|---|---|---|---|
| 梯度消失 | 浅层参数几乎不更新 | 链式法则连乘多个 <1 的导数（sigmoid） | ReLU、残差连接（ResNet）、LayerNorm |
| 梯度爆炸 | 参数剧烈震荡/NAN | 链式法则连乘多个 >1 的导数 | 梯度裁剪（Gradient Clipping） |

Megatron 训练大模型时**梯度裁剪是标配**。
