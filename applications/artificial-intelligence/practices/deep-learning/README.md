# 深度学习基础实践

> 从基础神经网络到现代深度学习架构

## 📚 学习内容

### 01 - 卷积神经网络 (CNN)
**概念：** 专门处理网格结构数据（如图像）的神经网络

**实现内容：**
- 卷积层（Convolution）：特征提取
- 池化层（Pooling）：降维
- 经典架构：LeNet、简化版 ResNet
- 图像分类实战

---

### 02 - 循环神经网络 (RNN/LSTM)
**概念：** 处理序列数据的神经网络

**实现内容：**
- RNN 基础：循环结构
- LSTM：长短期记忆
- 序列预测、文本生成
- 时间序列分析

---

### 03 - Embedding 技术
**概念：** 将离散符号映射到连续向量空间

**实现内容：**
- Word Embedding：Word2Vec、GloVe
- Item Embedding：推荐系统基础
- 语义相似度计算
- 可视化降维（t-SNE）

---

### 04 - 优化与正则化技术
**概念：** 训练深度网络的技巧

**实现内容：**
- 优化器：SGD、Momentum、Adam
- 学习率调度：衰减、Warm-up
- 正则化：Dropout、Batch Normalization
- 梯度裁剪、权重初始化

---

## 🎯 学习目标

通过这些实践，你应该能够：
- ✅ 理解 CNN 处理图像的原理
- ✅ 掌握 RNN/LSTM 处理序列的方法
- ✅ 理解 Embedding 的作用（推荐系统核心）
- ✅ 掌握深度网络训练技巧
- ✅ 为学习推荐系统打好基础

## 💡 为什么这个顺序？

```
机器学习基础 → 深度学习基础 → 推荐系统

理由：
1. CNN → 理解卷积、特征提取（用户/物品特征）
2. RNN/LSTM → 理解序列建模（用户行为序列）
3. Embedding → 理解表示学习（推荐系统核心技术）
4. 优化技巧 → 训练大型模型必备

推荐系统 = 深度学习技术的综合应用
```
