# 深度学习框架

> 主流深度学习框架的学习与对比

## 📚 框架列表

### [TensorFlow](./tensorflow/)
- **维护者**：Google
- **特点**：完整的端到端平台，支持移动和生产部署
- **适用场景**：工业级部署、移动端应用、大规模分布式训练
- **学习状态**：✅ 理论部分已完成（2025-12-17）
  - ✅ 核心概念、Keras API、数据管道、训练优化

### PyTorch（计划中）
- **维护者**：Meta
- **特点**：动态计算图，研究友好，生态丰富
- **适用场景**：研究原型、计算机视觉、NLP

### JAX（计划中）
- **维护者**：Google
- **特点**：NumPy + 自动微分 + GPU/TPU，函数式编程
- **适用场景**：研究、高性能计算、科学计算

## 🔍 框架对比

| 特性 | TensorFlow | PyTorch | JAX |
|------|-----------|---------|-----|
| 学习曲线 | 中等 | 平缓 | 陡峭 |
| 部署 | 优秀 | 良好 | 较弱 |
| 研究 | 良好 | 优秀 | 优秀 |
| 社区 | 大 | 大 | 中 |
| 编程风格 | 声明式 | 命令式 | 函数式 |

## 🎯 学习建议

1. **初学者**：从 TensorFlow/Keras 或 PyTorch 开始
2. **研究者**：PyTorch 或 JAX
3. **工程师**：TensorFlow（更成熟的部署工具链）
4. **科学计算**：JAX（与 NumPy 最接近）
