# 训练：梯度下降与优化

## 一句话

训练 = 反复调整参数，让损失函数越来越小。

---

## 梯度下降的三种变体

### 符号约定

- N：总样本数
- B：batch size
- i：样本索引

### 批量梯度下降（BGD）

每次用**全部 N 个样本**算梯度。

```
θ = θ - η · 1/N · Σᵢ ∇Lᵢ(θ)
```

**缺点**：N 很大时每步太慢。

### 随机梯度下降（SGD）

每次用**1 个样本**算梯度。

```
θ = θ - η · ∇Lᵢ(θ)
```

**缺点**：梯度噪声大，收敛路径震荡。

### 小批量梯度下降（Mini-batch GD）

每次用**一个 batch**（B 个样本）算梯度。

```
θ = θ - η · 1/B · Σᵢ∈batch ∇Lᵢ(θ)
```

**实际最爱用**：Batch size=32/64/128，平衡效率和稳定性。

---

## 学习率

最关键的超参数。

| 学习率 | 效果 |
|---|---|
| 太大 | 跳过最优点，loss 震荡发散 |
| 太小 | 收敛极慢，卡在局部最优 |
| 合适 | 稳定下降，快速收敛 |

**实践中**：用学习率调度（warmup + cosine decay），像 Megatron 那样。

---

## 优化器进化

| 优化器 | 核心思想 | 效果 |
|---|---|---|
| SGD | 固定学习率 | 基础 |
| SGD+Momentum | 加惯性项，累计历史梯度方向 | 加速收敛，减少震荡 |
| Adam | Momentum + 自适应学习率（每参数不同步长） | 鲁棒，好调参 |
| AdamW | Adam + 解耦权重衰减 | 泛化更好，LLM 标配 |

Megatron 用的就是 **AdamW**。

---

## 正则化

防止过拟合的手段。

| 方法 | 做法 | 效果 |
|---|---|---|
| L2（Weight Decay） | 损失加 λ·‖w‖² | 让权重趋向小值，模型更平滑 |
| L1（Lasso） | 损失加 λ·|w| | 让部分权重变 0，特征选择 |
| Dropout | 训练时随机丢弃部分神经元 | 强迫模型不依赖某一个特征 |
| 数据增强 | 对输入做变换（翻转、旋转等） | 变相增大训练集 |
