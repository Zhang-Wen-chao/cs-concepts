# 线性模型

## 一句话

假设输入和输出之间是**直线关系**，找一条最好的线来拟合数据。

---

## 线性回归（回归问题）

### 模型

```
y = w·x + b
```

- x：输入特征（向量）
- w：权重（每个特征的重要程度）
- b：偏置
- y：预测值（连续实数）

### 损失函数：MSE

```
L = 1/N · Σ(ŷ - y)²
```

MSE 越大说明预测越离谱。

### 梯度

```
∂L/∂w = 2/N · Σ(ŷ - y)·x
∂L/∂b = 2/N · Σ(ŷ - y)
```

用梯度下降更新 w、b。

---

## 逻辑回归（二分类问题）

### 模型

```
p = σ(w·x + b)  其中 σ(z) = 1 / (1 + e⁻ᶻ)
```

输出 0~1 之间的概率，≥0.5 判为正类。

### 损失函数：交叉熵

```
L = -[y·log(p) + (1-y)·log(1-p)]
```

为什么不用 MSE 做分类？因为 MSE 对偏离决策边界的样本惩罚不够狠，且梯度会饱和。

---

## 线性 vs 逻辑回归

| | 线性回归 | 逻辑回归 |
|---|---|---|
| 输出 | 连续值（房价 100 万） | 概率（垃圾邮件 0.95） |
| 激活 | 无（恒等） | Sigmoid |
| 损失 | MSE | 交叉熵 |
| 决策边界 | 无 | 线性 |

---

## 局限性

线性模型只能学**直线/平面**决策边界。面对非线性数据（如圆圈分布的两类点）无能为力——这时候就需要神经网络。
