# C++ RPC 项目实战

三道递进的 RPC 项目，从手写协议到工业级框架再到 ML 模型服务。

| 项目 | 路径 | 说明 |
|------|------|------|
| 01_simple_rpc | `practices/01_simple_rpc/` | 从零手写 RPC：TCP + 二进制协议 + 函数注册 |
| 02_brpc_hello | `practices/02_brpc_hello/` | 使用 bRPC 框架实现 Echo 服务 |
| 03_recommend_service | `practices/03_recommend_service/` | 基于 bRPC 的推荐模型服务 |
