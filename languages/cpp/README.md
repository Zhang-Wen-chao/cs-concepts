# C++ 学习路径

> 从零基础到能讲清机制、写出现代 C++、从容应对 C++ 面试

## 🗺️ 10 阶段路线图

| Stage | 主题 | 核心问题 | 目录 | 笔记数 |
|---|---|---|---|---|
| 0 | 编译底层 | 编译器到底做了什么？ | `00_compilation_fundamentals/` | 3 篇 |
| 1 | C++ vs Python 本质 | 静态类型、值语义、栈堆是什么？ | `01_cpp_vs_python/` | 自测 8 套 |
| 2 | 内存模型与 RAII | 谁负责释放内存？move 是什么？ | `02_memory_and_raii/` | 3 篇 + `_supplementary/` |
| 3 | 面向对象深入 | 虚函数表怎么工作？ | `03_oop_deep_dive/` | 4 篇 |
| 4 | 模板与泛型 | 模板怎么实例化？ | `04_templates/` | 4 篇 + `_supplementary/` |
| 5 | STL 源码级 | vector / unordered_map 怎么实现？ | `05_stl_internals/` | 4 篇 + `_supplementary/` |
| 6 | 现代 C++ 特性 | C++11/14/17/20 怎么用对？ | `06_modern_cpp/` | 5 篇 + `_supplementary/` |
| 7 | 并发与内存序 | 内存序是怎么回事？ | `07_concurrency/` | 5 篇 + `_supplementary/` |
| 8 | 系统编程 | 网络 I/O、序列化、内存管理、性能优化 | `08_system_programming/` | 4 篇 + practices |
| 9 | RPC 项目实战 | 手写 RPC → bRPC → 推荐服务 | `09_projects/` | 3 个项目 |

每个 stage 的 `_supplementary/` 子目录存放更详细的深度笔记和实践代码，作为主线笔记的补充参考。
`08_system_programming/` 和 `09_projects/` 是从旧 `_reference/` 迁移并整理后的新阶段。
